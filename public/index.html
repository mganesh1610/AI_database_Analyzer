What This Viewer Is Good At

            

              

                
Schema Understanding

                

                  Use the explorer, relationship map, and DDL pane to understand how tables, views,
                  and foreign keys fit together.
                

              

              

                
Data Inspection

                

                  Browse live rows with search, sort, pagination, and export controls without writing
                  custom SQL every time.
                

              

              

                
Column Profiling

                

                  Profile a selected column to see null counts, distinct counts, and the most common
                  values for quick sanity checks.
                

              

              

                
Safe Querying

                

                  The SQL Lab is intentionally read-only, so it helps explore data without risking
                  accidental changes.
